## NMR（Bruker）編

**目　標**：　NMRの代表的な測定方法である１次元スペクトルの処理の流れを習得することを目指します．  
**機　種**：　Bruker AV-200 
**データ**：　BrukerはフォルダーにFID, FFTデータが階層的に格納されます

対応機種 ：　
rawデータ：　BrukerはフォルダーにFID, FFTデータが階層的に格納されます．
